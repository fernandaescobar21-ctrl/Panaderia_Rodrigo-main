# 🍞 Panadería Rodrigo

Proyecto académico del Módulo 6 — Bootcamp Alkemy  
Sistema web de gestión interna para una panadería artesanal.

## 🚀 Tecnologías usadas
- Java 17 + Spring Boot 3.2.5
- Spring MVC + Thymeleaf
- Spring Data JPA + H2
- Spring Security
- Maven + Lombok

## ▶️ Cómo ejecutar
1. Clonar el repositorio
2. Abrir en IntelliJ IDEA
3. Ejecutar `RodrigoApplication.java`
4. Entrar a: `http://localhost:8080`
